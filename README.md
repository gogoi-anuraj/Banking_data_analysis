# Banking Data Analysis – Telemarketing Campaign

## Overview
This project performs **Exploratory Data Analysis (EDA)** on a banking dataset to understand customer behavior in telemarketing campaigns and identify factors influencing **term deposit subscriptions**. The project was completed as part of the **Finlatics Data Analytics Program**.

## Dataset
The dataset contains demographic, financial, and campaign-related information about bank clients contacted during a marketing campaign.

Key attributes include:
- Age
- Job
- Marital Status
- Education
- Balance
- Housing Loan
- Personal Loan
- Contact Type
- Campaign Details
- Term Deposit Subscription (Target Variable)

## Tools & Libraries
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  

## Analysis Performed
- Data cleaning and preprocessing  
- Age and job distribution analysis  
- Education and marital status insights  
- Loan and balance analysis  
- Campaign performance analysis  
- Term deposit subscription distribution  
- Correlation analysis between variables  

## Key Insight
Most clients **did not subscribe to term deposits**, and factors such as **contact duration, job type, and previous campaign outcomes** influence subscription probability.
